# Roadmap - План развития

## Бизнес-модель
- **Цена**: ~300 грн/месяц при условии более 10 клиентов в месяц
- **Целевая аудитория**: косметологи, массажисты, стоматологи, репетиторы, тренеры и другие специалисты, работающие по записи

---

## Высокий приоритет (реальная ценность)

### 1. Фото до/после
- [ ] Прикреплять фото к визиту клиента
- [ ] История прогресса для клиента
- [ ] Портфолио для привлечения новых клиентов
- [ ] Галерея в карточке клиента

### 2. Напоминания
- [ ] За день до визита (Telegram уведомление)
- [ ] "Пора на повторную процедуру" через N дней/недель
- [ ] Настройка интервала напоминаний для каждой услуги
- [ ] Уменьшает неявки

### 3. Карточка клиента с заметками
- [ ] Аллергии, противопоказания
- [ ] Предпочтения (персональные заметки)
- [ ] Что использовали в прошлый раз
- [ ] История всех визитов

### 4. Статистика и аналитика
- [ ] Доход за месяц/неделю/период
- [ ] Количество клиентов
- [ ] Средний чек
- [ ] Популярные услуги
- [ ] Графики и диаграммы

---

## Средний приоритет

### 5. Онлайн-оплата / предоплата
- [ ] Интеграция с Monobank
- [ ] Интеграция с LiqPay
- [ ] Предоплата при записи (уменьшает отмены)
- [ ] История платежей

### 6. Публичная страница специалиста
- [ ] Мини-сайт с услугами и ценами
- [ ] Кнопка "Записаться" → переход в бот
- [ ] Отзывы клиентов
- [ ] Фото работ (портфолио)
- [ ] Контактная информация

### 7. Протокол визита
- [ ] Фиксация: что делали, какие препараты, дозировки
- [ ] Рекомендации клиенту после процедуры
- [ ] Шаблоны для типовых процедур
- [ ] Отправка рекомендаций клиенту в Telegram

---

## Низкий приоритет (возможно в будущем)

### 8. Хранение документов
- [ ] Фото анализов, справок
- [ ] Информированное согласие
- Примечание: усложняет приложение, мало кто будет использовать

### 9. Программа лояльности
- [ ] Накопительные скидки
- [ ] Бонусные баллы
- Примечание: для соло-специалиста избыточно

### 10. Рассылки
- [ ] Рассылка по базе клиентов
- [ ] Акции и спецпредложения
- [ ] Поздравления с днем рождения
